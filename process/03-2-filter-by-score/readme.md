# プロセス 03-2-filter-by-score: スコアによるフィルタリング

## 概要
02工程のCSVの各行を03-1工程のスコア結果を参考にしてフィルタリングし、スコアが閾値以上の食品のみを残すプロセスです。
フィルター閾値は`index.ts`の`SCORE_THRESHOLD`定数で変更できます（デフォルト: 6）。

## 入力
- `../02-food-name-normalize/result/claude-json-header-with-structured-names.csv` - 02工程で構造化されたCSVデータ
- `../03-1-dog-food-scoring/result/scores.csv` - 03-1工程で生成されたスコアデータ

## 処理内容
- スコアファイルを読み込み、スコアが閾値以上の食品番号のセットと理由・スコアのマップを作成
- 02工程のCSVファイルを読み込み、各行の食品番号（2列目）を確認
- スコアが閾値以上の食品番号に該当する行のみを残してフィルタリング
- 各フィルタリングされた行に「理由」と「スコア」の列を追加
- フィルタリング後のCSVファイルとして出力

## 出力
- `result/filtered-by-score.csv` - スコア閾値以上の食品のみが残され、理由とスコアの列が追加されたCSVファイル
