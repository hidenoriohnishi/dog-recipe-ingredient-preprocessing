# 04-1: 成分表マージ

## 概要

3つの成分表を「食品番号」をキーとしてマージし、レシピ生成に必要な列のみを含む新しい表を作成する。

## 入力

1. `03-2-filter-by-score/result/filtered-by-score.csv` - フィルタ済み食品表（ベース）
2. `04-0/amino_acid_composition.csv` - アミノ酸成分表
3. `04-0/fatty_acid_composition.csv` - 脂肪酸成分表

## 出力

`result/merged-nutrition.csv` - マージ済み成分表

## マージ仕様

`04-0/plan.json`に従い、各テーブルから以下の列を抽出してマージする：

### filtered_food_table から
- 食品番号、食品名
- 水分、たんぱく質、脂質、食物繊維総量、灰分
- カルシウム、リン、ナトリウム
- レチノール、ビタミンD
- ヨウ素、セレン
- 構造化食品名、理由、スコア

### amino_acid_table から
- イソロイシン、ロイシン、リシン
- メチオニン、シスチン、含硫アミノ酸
- フェニルアラニン、チロシン、芳香族アミノ酸
- スレオニン、トリプトファン、バリン
- ヒスチジン、アルギニン

### fatty_acid_table から
- 脂肪酸総量
- 多価不飽和脂肪酸、n-3系多価不飽和脂肪酸、n-6系多価不飽和脂肪酸
- リノール酸、α-リノレン酸
- EPA、DHA、アラキドン酸
